🌦️ Basic Weather App (Tkinter)

📖 Description

This is a small desktop application that displays the current weather of any city entered by the user.

The application is built with the help of the Python programming language and the Tkinter library for the interface and the Open Weather Map API for the live weather updates.

✨ Features

The application has a clean and simple interface.

The application fetches live updates of the weather.

The application displays the city name, country, temperature in Celsius, and the current weather.

The application also displays error messages for invalid city names.

🛠️ Technologies Used

Python 3

Tkinter

Requests Library

Open Weather Map API.

⚙️ How the System Works

The user enters a city name into the input box.

When the user clicks the "Search to know Weather" button, the application sends a request to the OpenWeatherMap API.

The API response is processed and the following information is displayed on the screen:

Location (City and Country)

Temperature (in Celsius)

Weather description (e.g., Clear sky, Rain, Clouds)

If the city name is not found, the program displays an error popup.
